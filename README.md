# Arquiteto de migração de infraestrutura de nuvem Oracle profissional - 2025

A Certificação Oracle Cloud Infrastructure Migration Architect Professional foi desenvolvida para arquitetos de migração envolvidos na migração de cargas de trabalho para OCI. É recomendado que os candidatos possuam uma compreensão de ambientes baseados em VMware no local e implantações de banco de dados no local. O candidato alvo deve ter 6 meses ou mais de experiência prática com Oracle Cloud Infrastructure.

Após concluir este curso, você será capaz de:

 - Planeje e execute migrações de cargas de trabalho de ambientes locais para o Oracle Cloud Infrastructure (OCI).
 - Projete e configure redes OCI para garantir conectividade segura e de alto desempenho para cenários de migração.
 - Migre cargas de trabalho locais para o OCI e o Oracle Cloud VMware Solution (OCVS).
 - Mova aplicativos em contêineres para o OCI Container Engine for Kubernetes (OKE).
 - Transferir cargas de trabalho entre regiões OCI.
 - Migre bancos de dados Oracle para OCI usando práticas recomendadas para garantir alta disponibilidade, segurança e integração perfeita.

Benefícios para você

 - Obtenha uma compreensão clara dos princípios fundamentais e das melhores práticas para planejar e executar migrações de carga de trabalho sem interrupções, identificando estratégias-chave em projetos de transformação de nuvem.
 - Desenvolva experiência prática na aplicação de habilidades essenciais em redes, migração de banco de dados e mobilidade de carga de trabalho por meio de demonstrações práticas.
 - Fortaleça seu conhecimento básico sobre ferramentas de migração OCI e melhores práticas do setor analisando cenários do mundo real em arquitetura de nuvem e estratégia de migração.
 - Melhore sua capacidade de avaliar e otimizar o desempenho, a segurança e a eficiência de custos em ambientes de nuvem, preparando-o para funções de alta demanda na transformação da nuvem.
